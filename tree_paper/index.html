Overview

Urban tree inventories are essential for sustainable planning, biodiversity, and climate resilience, but conventional tree cadaster mapping remains labor-intensive and spatially limited.

This study proposes a semi-automatic pipeline that combines open urban tree inventories, LiDAR-derived canopy height models, and multispectral aerial imagery to generate dense, high-quality labels for deep learning models.

Methodology

Sparse seed set

CHM-derived crowns filtered by official cadasters form the first training set.

Dummy model

A preliminary detector generates dense pseudo-labels across Dresden.

Manual refinement

Human correction turns pseudo-labels into a dense annotated dataset.

Model comparison

Five object detectors are evaluated across six input band combinations.

Results

Best inputsRGB, NIR, and CHM
PerformancemAP50 > 0.79

Anchor-free models such as ATSS and FSAF performed best, and the refined dense labels improved transferability to Leipzig as an independent test site.
