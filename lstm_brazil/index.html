Overview

This study presents the first comprehensive assessment of LSTM-based streamflow modeling across Brazilian catchments using the CABra dataset. We evaluate country-scale and hydrologically grouped strategies with multiple dynamic input configurations and cross-validation schemes.

The results show that country-scale models generally perform better in humid regions and large catchments, while performance declines in arid areas with non-perennial rivers. Increasing the number of dynamic inputs does not always improve performance, suggesting possible multicollinearity issues.

Methodology

Country-scale

Five-fold cross-validation across all basins with repeated random seeds.

Hydrological groups

Separate models trained for catchments with similar hydrological behavior.

Leave-one-group-out

Independent testing by withholding one hydrological group at a time.

Input evaluation

Four dynamic input combinations to examine model sensitivity and transferability.
