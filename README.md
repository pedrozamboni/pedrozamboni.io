# pedrozamboni.io

This repository is the source for the GitHub Pages site at pedrozamboni.io.

## Structure

- `index.html` is the landing page.
- `pond/`, `stereo_camera/`, `tree_paper/`, and `lstm_brazil/` are individual project pages.

## Publishing

1. Push changes to the repository.
2. In GitHub Pages settings, publish from the default branch and the repository root.
3. Keep project links relative so the pages work from the root site.
