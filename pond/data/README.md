# Water Ponding Analysis Dataset

This directory contains the dataset used in our research on water ponding analysis.

## Dataset Structure

```
data/
├── images/         # Time-lapse images of soil plots
├── annotations/    # Ground-truth annotations
├── measurements/   # Runoff and soil moisture measurements
└── models/        # Trained model weights and configurations
```
