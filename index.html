Ongoing Projects

Automated fusion and classification of ALS and UAV data for heavy rain hazard maps

This project aims to merge both DEM types with a robust method for aligning point clouds and automated integration of UAV-recognized structures, enhancing accuracy and coverage for runoff simulations to provide municipalities with up-to-date, informative heavy rain risk maps.

Project link

Deep Learning-Driven Mapping of Individual Urban Trees

This project focuses on mapping individual trees in urban environments using deep learning and high-resolution multimodal data, such as aerial imagery, LiDAR, and multispectral inputs. The system can accurately detect, classify, and localize trees at the individual level, supporting urban planning, green space monitoring, and environmental analysis. Link to the project

AI4Flow-BR: Training Strategies for Multi-Scale Discharge Estimation

This project develops and evaluates training strategies for applying LSTM and Transformers models to river discharge estimation across Brazil, combining deep learning techniques with hydrological modeling while fostering the next generation of researchers. Link to the project

Deep learning for river water velocity and discharge estimation

This project uses AI to estimate river water velocity and calculate discharge by learning patterns directly from image sequences. The approach reduces the need for manual adjustments and adapts better to varying environmental conditions.

AI and 3D Photogrammetry for Flower Mapping

This project combines AI and UAV imagery to detect and analyze flowers in trees at both landscape and individual tree levels, creating a scalable, precise system for floral monitoring using AI-driven detection and 3D photogrammetric analysis.

AI and Stereo Photogrammetry for Low-Cost River Digital Twin Generation

This research proposes using a stereo camera setup to directly create 3D river cross-sections from image pairs, eliminating the need for pre-existing models. The innovation combines AI with stereo-photogrammetry to improve image matching and handle 3D reconstruction from raw images. Link to the project

Counting Eucalyptus Trees with AI and Fisheye Photogrammetry

This project explores the use of low-cost fisheye cameras combined with AI and photogrammetry to detect and count individual eucalyptus trees, offering an efficient and affordable method for forest inventory and monitoring.
